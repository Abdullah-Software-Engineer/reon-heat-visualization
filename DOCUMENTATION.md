## üîç Quick Reference

### Key Concepts

- **Heatmap Visualization**: Interactive chart showing runtime sources over time
- **Date Range Filtering**: Filter data by custom date ranges
- **Real-time Polling**: Automatic data updates at configurable intervals
- **Data Export**: Download visualizations and raw data

### Key Technologies

- React 19.2.0
- TypeScript 5.9.3
- ECharts 5.4.3
- Tailwind CSS 4.1.17
- Vite 7.2.2

### Key Components

- `Heatmap` - Main visualization component
- `DateRangePicker` - Date filtering UI
- `DownloadButton` - Export functionality
- `Dropdown` - Reusable dropdown menu

### Key Hooks

- `useRuntimeData` - Data fetching with polling
- `useClickOutside` - Outside click detection
